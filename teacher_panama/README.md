# TeacherPanama Pro — PWA Combinada (CSV/PDF + v2 SW)

Sube esta carpeta como `teacher_panama/` a tu repositorio.

Incluye:
- `index.html` con botones CSV/PDF y funciones integradas (exportación de calificaciones y asistencia a CSV; reporte PDF con jsPDF + autoTable).
- `service-worker-v2.js` (cachés nuevos, `network-first` para navegaciones) y registro en `index.html`.
- `manifest.json` ajustado al scope `/teacher_panama/`.
- `icons/` (192 y 512).

Pasos:
1. Sube todo a `teacher_panama/`.
2. Espera 1–2 minutos por el despliegue.
3. Abre `https://<usuario>.github.io/teacher_panama/?v=2` para forzar recarga.
4. (Opcional) Desinstala la PWA anterior y vuelve a instalar.
